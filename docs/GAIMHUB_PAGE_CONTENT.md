## Overview

Vector Maniac is a fast-paced mobile arcade game featuring:
- 44+ unique spacecraft with distinct abilities
- 15+ procedurally generated enemy types
- Endless survival and arena battle modes
- Deep progression with upgrades and companions

Built using a **human-led AI automation pipeline** — combining Lovable for code generation, Suno for audio, and iterative AI-assisted design refinement.

---

## What I'm Testing

### Human-Led AI Development
This project explores the frontier of AI-assisted game development:

**Code Generation** — Using Lovable to rapidly prototype and iterate on React/TypeScript codebase

**Audio Creation** — Soundtrack and sound effects generated with Suno AI

**Art Direction** — Human-curated AI-generated visual concepts

**Iterative Refinement** — Continuous human feedback loops with AI execution

### Key Learnings
- AI accelerates implementation but requires strong creative direction
- Iteration cycles are 10x faster than traditional development
- Human judgment remains essential for game feel and balance

## Why World

### The Problem
Mobile games suffer from:
- Bot-farmed rewards and fake accounts
- Manipulated leaderboards
- Multi-account exploitation

### The Solution
World ID enables **proof-of-personhood** for gaming:

**Human-Verified Leaderboards**
Every entry proven to be a unique human player.

**Fair Daily Rewards**
One bonus per person per day, guaranteed.

**Sybil-Resistant Competitions**
Arena battles between verified players only.

### Integration Plan
1. **World ID Authentication** — Link player progress to verified identity
2. **WLD Payments** — Cosmetic purchases using WLD tokens
3. **World Mini App** — Native deployment in World ecosystem
