## Screenshot Guidelines

When capturing screenshots:

1. **Resolution**: Capture at 1920x1080 or device native resolution
2. **Format**: PNG for quality, JPEG for smaller file size
3. **Naming**: Use the format `XX-description.png`
4. **Content**: Avoid capturing personal data or debug information
