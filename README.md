# Galactic Overdrive

A fast-paced space arcade game with multiple game modes and ships.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Mobile App (Capacitor)

```sh
# Sync to iOS/Android after build
npx cap sync

# Run on iOS
npx cap run ios

# Run on Android
npx cap run android
```
